# WebAppRunner

Extend your web application with android features

* Signature
* Vibration
* Location
* Barcode
* NFC
* Push notification

## Info

It is a study project. I develop in Sony Xperia L and Samsung Galaxy A3 with [AIDE](https://www.android-ide.com/)
